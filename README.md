# phpsls
PHPSLS is a Secure Login System for any website

## features

- Secure Session
- CSRF Tokens
- SQL Injection protection
- Anti Spam reCaptcha
- Registration with Email Verification
- Forgot password
- and more ... 

## installation 

- add account/ folder to your script and edit config.php file

## required

- PHP version >= 5.4
- Mysqli
- Mail function enable
